# SFTP

## 打开 SFTP 页面

从当前 SSH 会话打开 SFTP。页面同时提供远程文件浏览、本地文件浏览和该会话的传输状态。

## 本地目录记忆

对于已保存的 SSH 会话，本地文件浏览器会恢复该会话最后一次成功打开的目录。新建或未保存的连接始终从用户主目录开始，且不会记录本地目录。如果已记忆的目录被删除或无法读取，AxShell 会打开用户主目录，但不会覆盖原有记录。

这些本地路径只保留在当前电脑，不会进入 WebDAV 或 S3 会话同步。

## 文件操作

- 浏览远程目录，并显示或隐藏隐藏文件。
- 对已加载条目排序，并通过路径导航。
- 上传文件或文件夹。
- 下载文件或文件夹；目录下载会在适合时使用临时归档。
- 新建文件夹并递归删除选中路径。
- 使用系统默认编辑器打开远程文件，保存后自动上传修改。
- 预览支持的文件和受限目录内容。

## 超大目录

远程列表按需加载，不会一次把整个目录读入界面。

- 每页最多追加 250 个条目。
- 仍有条目时可以使用**加载更多**。
- 单个列表最多保留 2,000 个条目，或 2 MiB 的名称/路径数据。
- 达到安全预算后，AxShell 会停止加载并显示截断状态，不会把结果误报为目录结束。

## 传输任务

传输任务支持暂停、恢复和取消。传输历史显示进行中、完成、失败和中断状态，最多保留 100 条记录。

存在活动任务时关闭 SFTP 页面会使用已配置的确认流程，避免静默丢弃正在执行的传输。
